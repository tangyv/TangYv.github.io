第四章 检视阅读

精通基础阅读：检查生字、文法、文章结构

检视一 系统性略读skim或粗读pre-read

建议

  1. 书名页 特别注意副标题、特殊角度
    归类，那个类型含有哪些书
  2. 研究目录页
    论说性类书目
    书想要做什么
  3. 索引
    主要：论说类
    评估：议题范围、书籍种类、作者、判别重要词汇
  4. 新书书衣
    出版者介绍
    广告性文案-偏颇-吹牛、无话可谈

此时，足以判断是否继续阅读

  1. 挑选主题相关篇章阅读

  2. 随便翻阅读
    连续段落+连续页，跳读全书
    注意主要论点、留意主题基本脉动、不要忽略最后2-3页

    作者喜欢在最后几页将自己新颖观点从新总结

此时，完成系统略读

要求，积极主动。

检视二 粗浅的阅读

对的方向
第一次阅读难点跳过，先读完全书 50%或更少
故事性莎士比亚剧本，逐字逐句，忘记开头

​ 论说性 《国富论》为一般读者写的书，大原则

阅读的速度

量子波动速读 :sweat_smile:

if you can read slowly, you can read quickly.

不只是因为你读的快，只读一小部分：不同方式、不同目标
快慢有当

半出声：最初学会阅读之后几年
眼睛活动：逗留flxate，移动时看不见，只有停下来才能看见，倒退regress原点
矫正工具：遮挡移动法
